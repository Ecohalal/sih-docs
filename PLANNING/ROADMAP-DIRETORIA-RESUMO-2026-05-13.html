Sys Halal — Nova versão da API de integração externa (v2) Evolução da API hoje usada por parceiros e pelo importador de arquivos TXT. A versão atual empurra detalhes de produto, datas de abate e de processamento para campos genéricos de "descrição" e "informações adicionais", herdados da versão anterior do sistema. A base de dados já está preparada com as colunas segregadas; a API e o importador de TXT precisam evoluir para popular as colunas certas e devolver os dados estruturados. Habilita certificados de embarque novos com rastreabilidade halal completa lote a lote. Inicia em paralelo com as outras frentes.
